# Threat_Detection_Alarming_System

This project is a reasoning engine developed using Bayesian Networks to detect abnormal events such as illegal border crossings.

## Features
- Real-time alarm system
- Detection of abnormal behavior
- Visualization of individuals and groups on a map
- Probabilistic reasoning using Bayesian Networks

## Technologies
- Python
- PostgreSQL
- Machine Learning
- Bayesian Networks

## Description
This project was developed as a final year project for a BSc Computer Science degree at the University of Reading (2025).
